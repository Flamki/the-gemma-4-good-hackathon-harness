# Kaggle Submission Checklist

## Required for valid submission
1. Kaggle writeup completed (under 1,500 words)
2. Track selected
3. Public YouTube video attached (<= 3 minutes)
4. Public code repository link attached
5. Public live demo URL or files attached
6. Media gallery cover image added
7. Draft switched to submitted before deadline

## Quality checks before final submit
1. Readme includes setup and architecture
2. Demo video shows real app execution
3. Writeup clearly explains how Gemma is used
4. Repo has reproducible run steps
5. Identity verification completed
6. If using a temporary tunnel URL, verify it is reachable when judges open it
